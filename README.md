# Religious Cognition Collaborative Website

Responsive, multi-page research website for the Religious Cognition Collaborative.

## Local preview

Run `npm run build`, then serve the `public` directory with any static web server.

## Vercel

Import this repository into Vercel. The checked-in `vercel.json` sets the build command, output directory, clean URLs, and route rewriting.

## Before public launch

Replace the temporary contact email if needed; add the sponsoring nonprofit's approved legal name and logo; and populate only verified project, team, publication, funding, and donation information.
