# Deployment

Import this repository into Vercel. The root-level `vercel.json` extracts and builds the packaged Next.js website automatically.

The source package is `religious-cognition-collaborative-site.zip`. It contains the editable Next.js project in the `rcc-site` directory.

Before public launch, replace the temporary contact email, add the sponsoring nonprofit's approved legal name and logo, and populate only verified project, team, publication, and donation information.
