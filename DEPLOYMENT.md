# Vercel Deployment

1. In Vercel, choose **Add New → Project**.
2. Import `waypointinstitute/app` from GitHub.
3. Leave the root directory at the repository root.
4. Deploy. Vercel will use `vercel.json` automatically.

The build concatenates the checked-in source fragments into `public/site.js` and publishes the `public` directory.

Before launch, verify the contact email, sponsoring nonprofit identity and logo, project details, team listings, publications, and donation routing.
